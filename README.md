# зачем?

главная проблема данной тв приставки (в частности версии, которая поставляется от казахтелекома) нету нормальной возможности установить свои приложения. файлового мененджера с коробки нету, магазин приложений даёт установить только rutube. конечно, можно поднять магазин приложений, но это слишком долгое занятие.
цель скрипта - автоматизировать установку файлового менеджера путём установки приложения по ssh.

# как использовать?

в первую очередь, нужно ***обязательно*** включить ssh доступ у тв приставки. для этого необходимо включить режим разработчика в настройках
1. переходим в настройки, зажимаем кнопку f на пульте. после пары секунд вылезет окно как ране
2. заходим в пункт "об устройстве". нажимаем 5-7 раз на пункте с "сборка".
3. также, стоит записать серийный номер + ip-адрес тв приставки. он находится в пункте "статус" в "об устройстве"
4. возращаемся назад. заходим в пункт "режим разработчиков" и листаем вниз пока не найдём пункт "SSH". включаем его

дальше, запускаем скрипт.
`https://raw.githubusercontent.com/mryoshiii/mdtkbreak/refs/heads/main/mdtkbreak.sh | bash mdtkbreak.sh"

следуем по инструкции в терминале. если всё прошло правильно, то в конце должно вылезти:
```
Success
sh: can't find tty fd: No such device or address
sh: warning: won't have full job control
NV711KZK:/ # 
```
поздравляем! значит, файловый менеджер был установлен.
